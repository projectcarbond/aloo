An easy starting point with ReactJS, SASS and Redux with Server Side Rendering.

## Installation

```bash
  $ git clone https://github.com/projectcarbond/aloo.git [your_project_name]
  $ cd [your_project_name]
  $ yarn install
```

## Development

```bash
  # Develop
  $ yarn start

  # All Tests
  $ yarn test

  # Unit Test
  $ yarn unit-test

  # Lint
  $ yarn lint
```
